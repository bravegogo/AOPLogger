# AOPLogger 切面日志
###
## Installation

### via CocoaPods
Install CocoaPods if you do not have it:-
````
$ [sudo] gem install cocoapods
$ pod setup
````
Create Podfile:-
````
$ edit Podfile
platform :ios, '5.0'


pod 'AOPLogger',  '~> 1.0'
pod 'Aspects'

$ pod install
````
Use the Xcode workspace instead of the project from now on.
